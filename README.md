# Python-Learn: Chuyển đổi Tư duy từ Java sang Python

Dự án này là một Dashboard học tập tương tác, được thiết kế đặc biệt để giúp các lập trình viên Java chuyển sang làm chủ hệ sinh thái Python và xây dựng ứng dụng Fullstack chuyên nghiệp.

## 🚀 Tính năng chính

- **Lộ trình 8 tháng:** Chia nhỏ kiến thức từ Foundation, OOP, Backend (FastAPI/Django) đến DevOps.
- **Tài liệu chuyên sâu:** Hướng dẫn chi tiết với bảng so sánh trực tiếp "Python Way vs Java Equivalent".
- **Git Tracker:** Theo dõi lịch sử commit (Conventional Commits) và đo lường sự tiến bộ.
- **Ví dụ thực chiến:** Code samples có thể chạy ngay và bài tập tư duy sau mỗi buổi học.

## 🛠️ Công nghệ sử dụng

- **Frontend:** TailwindCSS (Styling), Alpine.js (State management), Chart.js (Visualization).
- **Data:** JSON-driven architecture, LocalStorage.
- **Backend (Target):** FastAPI, SQLAlchemy, PostgreSQL.

## 📂 Cấu trúc dự án

- `/dashboard`: Giao diện Dashboard học tập.
- `/01-foundation` đến `/11-final-projects`: Nơi lưu trữ mã nguồn thực hành theo từng giai đoạn.
- `ARCHITECTURE.md`: Chi tiết về kiến trúc hệ thống và luồng dữ liệu.

## 📖 Bắt đầu

Mở file `dashboard/index.html` trong trình duyệt của bạn để bắt đầu hành trình học tập.
